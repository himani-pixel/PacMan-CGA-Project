# Pac-Man Game (Python & Pygame)

A classic **Pac-Man game built from scratch using Python and Pygame**.  
The project recreates core Pac-Man mechanics such as maze navigation, ghost movement, scoring, power-ups, lives, and win/lose conditions.

This project was developed for learning **game development fundamentals** and **computer graphics concepts**.

---

## Features
- Classic Pac-Man maze with pellets and power pellets  
- Smooth keyboard-controlled movement  
- Four ghosts with different movement behaviors  
- Power-up mode to eat ghosts and gain bonus points  
- Score and life system  
- Game Over and Victory screens  
- Restart option using keyboard input  
- Screen-wrapping tunnel movement  

---

## Controls
- **Arrow Keys** → Move Pac-Man  
- **Space Bar** → Restart game after Game Over or Win  

---

## Technologies Used
- Python 3  
- Pygame  
- Object-Oriented Programming
